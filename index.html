Welcome to the Austin Code Dojo!

We are a group of software craftsman who meet regularly for deliberate practice in the art of software design and construction. Our meetings cover a varied array of topics, including:

  • Test Driven Design
  • Build Automation
  • Cloud Technologies
and anything else we think is interesting and can help us learn to be better software developers.

If you want to hear about what's going on at the Dojo, check out our mailing list on Google Groups.We also host some code on GitHub, mostly examples of code we've worked on in meetings.

Meetings

We meet bi-weekly at DrillingInfo Corporate Headquarters in the Barton Creek/Zilker Park area. Our next meeting is on Wednesday, August 3. Please Sign Up so we know you're coming.
